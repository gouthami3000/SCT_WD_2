## 🚀 Project Description

This Stopwatch Web Application is developed using **HTML, CSS, and JavaScript**. It allows users to accurately measure time intervals with an intuitive and responsive interface.

The application includes all essential stopwatch functionalities such as starting, pausing, resetting, and recording lap times. It also features a modern glassmorphism user interface with smooth animations for an enhanced user experience.

---

## ✨ Features

- ▶️ Start Stopwatch
- ⏸️ Pause Stopwatch
- 🔄 Reset Stopwatch
- 🏁 Record Unlimited Lap Times
- ⏱️ Millisecond Accuracy
- 📱 Responsive Design
- 🎨 Modern Glassmorphism UI
- 🌈 Animated Gradient Background
- ⚡ Smooth Button Hover Effects
- ⌨️ Keyboard Shortcuts (S, P, R, L)
